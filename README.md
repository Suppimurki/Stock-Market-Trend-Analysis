# Stock-Market-Trend-Analysis

## Project Overview
Analyzed 5 plus years of historical stock market data to evaluate price movement, volatility, and risk patterns. Focused on identifying consistent trends and improving forecasting accuracy through structured data analysis.

## Objective
• Identify short term and long term price trends

• Measure volatility and risk using statistical metrics

• Evaluate stock performance based on return patterns

• Support investment decisions with data driven insights

## Tools and Technologies
• Excel for initial data cleaning and preprocessing

• SQL for data extraction, filtering, joins, and aggregations

• Python for analysis using Pandas and NumPy

• Power BI and Tableau for data visualization

## Data Collection and Preparation
• Collected historical stock data including open, close, high, low, and volume

• Cleaned datasets by handling missing values and removing duplicates

• Standardized date formats and validated data consistency

• Improved data quality and analysis accuracy by 20 percent

## Analysis Performed
• Conducted exploratory data analysis to understand price distribution and volume trends

• Calculated daily returns and moving averages to detect momentum

• Applied technical indicators such as Simple Moving Average and Relative Strength Index

• Measured volatility using standard deviation and return variance

• Compared multiple stocks to identify consistent performers

## Key Insights
• Identified trend reversal points using RSI and moving averages

• Detected high volatility periods during market corrections

• Observed correlation between trading volume spikes and price breakouts

## Outcome
• Developed visual reports and dashboards for clear trend interpretation

• Improved trend prediction accuracy by 20 percent through structured validation

• Enabled data backed evaluation of stock performance and risk levels
